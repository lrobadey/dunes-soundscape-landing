
## Dunes — Landing Page Plan

A cinematic, single-page landing experience for *Dunes*, Luca Robadey's EP for string quintet and electronics. The page leans into desert landscapes, dramatic typography, and a dark/warm color palette to match the "Desert / Cinematic" aesthetic.

---

### 1. Hero Section
- Full-bleed background with a placeholder dune image (easily swapped with your own photo later)
- Large, elegant title: **DUNES**
- Subtitle line: *EP for String Quintet and Electronics*
- Composer credit: *Luca Robadey*
- A soft cinematic overlay gradient (dark bottom fade) for text legibility
- Smooth scroll-down indicator

---

### 2. Overview / About the Work
- A centered prose section with the program note: 25-minute single-arc work, contemporary classical + EDM production, geomorphology inspiration
- Atmospheric background (subtle sand texture or dark tone)

---

### 3. Movements Section
Five cards or an elegant numbered list — one per movement, each tied to its dune location:
1. Simpson Desert — Australia
2. Badain Jaran Desert — China
3. Lençóis Maranhenses — Brazil
4. Sahara — Algeria
5. Great Sand Dunes — USA

Each entry displays the movement number, desert name, and country with tasteful typographic treatment.

---

### 4. Event Details
- Venue: Studio G, SFCM
- Date: May 2, 2026
- Format: Live performance with lighting and projections; premiere and digital release
- Supported by SFCM's Professional Development and Engagement Center
- Clean card or banner layout with an "Add to Calendar" link

---

### 5. RSVP / Notify Me Section
- A simple email capture form: name + email + "Notify Me" button
- Stores entries in local state for now (backend can be connected later for real persistence)
- Success confirmation message after submission
- Note about digital release notification

---

### 6. Footer
- Composer name + links placeholder (social/streaming) for post-release
- Copyright line

---

### Design Details
- **Color palette**: Deep charcoal/near-black backgrounds, warm sandy amber/gold accents, cream text
- **Typography**: A high-contrast serif for headings (cinematic feel), clean sans-serif for body
- **Responsive**: Fully mobile-friendly
- **Placeholder images**: High-quality dune stock photos in place — you can upload your own assets at any time
